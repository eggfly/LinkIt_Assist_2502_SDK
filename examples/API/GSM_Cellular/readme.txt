This example shows how to get the current and the neighboring cell information.

In this example, it will outputs the cell information in the Monitor tool. Before launching this example, a SIM card needs to be inserted and the GSM antenna needs to be attached to the GSM ANT. After launching the example,
the AT commands can be used through the Monitor tool to control the flow as below:

AT+[1000]Test01: opens the cell.
AT+[1000]Test02: shows the current cell information.
AT+[1000]Test03: shows the total number of the neighboring cells and all cell info.
AT+[1000]Test04: stops the cell.
